Hal ini yang mendasari saya untuk bisa belajar programming namun tetap sesuai dengan "pace" saya.

Saat disetujui, saya diberikan sepucuk e-mail berisi Host FTP, Database dan Password yang saat itu saya sangat tidak tahu apa fungsinya hingga akhirnya selama hampir 2 bulan saya mencari cara untuk memanfaatkan data tersebut. Akhirnya seelah 2 bulan saya berhasil menginstall Wordpress di Web Himpunan, dan beberapa bulan setelahnya Website tersebut sudah sangat aktif dengan berbagai kegiatan kemahasiswaan yang ada di Web tersebut.
Sejak saat itu saya mendalami bidang Website dan sangat tertarik dengan pengembangannya. Tahun ke-4 saya mulai mempelajari Python diselingi oleh Javascript karena saya merasa tidak puas dengan Wordpress dan PHP. Hingga saat ini saya masih mempelajari Python dan JS dari berbagai video di Youtube dan materi di situs-situs pembelajaran seperti FreeCodeCamp dan OdinProjects.
